# Weavr

A Ruby client for Apache Ambari (REST client)[https://github.com/apache/ambari/blob/trunk/ambari-server/docs/api/v1/index.md]

## Installation

Add this line to your application's Gemfile:

    gem 'weavr'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install weavr
